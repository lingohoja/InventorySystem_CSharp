# 進銷存專題：現代化 C# WinForm 系統

這是我的進銷存專題開發紀錄。目前系統已具備完整的商務邏輯與低耦合的專業架構。

## 📜 專案進度日誌

### 🚀 V3 版本：系統架構重構與採購閉環 (最新版)
**狀態：已完成**
- **UI 樣式封裝 (UIHelper)**：抽離按鈕、表格與 Placeholder 樣式，實現全系統一鍵換裝，落實 DRY (Don't Repeat Yourself) 原則。
- **資料庫邏輯封裝 (DBHelper)**：導入 `TableConfig` 元數據管理，並統一封裝「關聯刪除防呆」與「商務交易 (Transaction)」邏輯。
- **採購與庫存連動**：實作採購作業模組，完成「採購入庫加庫存、銷貨結帳減庫存」的完整資料流閉環。
- **購物車進階邏輯**：實作相同品項自動合併、庫存動態檢查，以及購物車單筆移除功能。

---

### 🗄️ V2 版本：資料庫整合與功能強化
**狀態：存檔於 `v2-legacy` 分支 (可選)**
- **資料庫交易 (Transaction)**：實作銷貨結帳與庫存同步扣除，確保資料一致性。
- **動態表單系統**：開發通用 `EditForm`，可根據 SQL 資料表結構自動生成輸入介面。
- **照片存儲優化**：改用 `VARBINARY(MAX)` 將產品相片直接存入 SQL Server。
