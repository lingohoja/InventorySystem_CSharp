# 進銷存專題：現代化 C# WinForm 系統

這是我的進銷存專題開發紀錄。目前已完成核心功能開發，並區分為 V1 與 V2 兩個重要階段。

## 📜 專案進度日誌

### 🚀 V2 版本：資料庫整合與功能強化 (最新版)
**狀態：已完成**
- **資料庫交易 (Transaction)**：實作銷貨結帳與庫存同步扣除，確保資料一致性。
- **動態表單系統**：開發通用 `EditForm`，可根據 SQL 資料表結構自動生成輸入介面。
- **照片存儲優化**：捨棄檔案路徑，改用 `VARBINARY(MAX)` 將產品相片直接存入 SQL Server，解決路徑失效問題。
- **防呆機制**：加入空值偵測與庫存不足預警。
- **介面預覽**：在產品維護頁面新增產品相片即時預覽功能。

---

### 🏗️ V1 版本：初始 UI 佈局與基本架構
**狀態：存檔於 V1 分支**
- **現代化介面設計**：完成側邊導覽欄與動態 `Panel` 切換邏輯。
- **基本 CRUD 介面**：完成客戶、產品與供應商的資料顯示區塊。
- **系統維護工具**：實作資料庫備份與還原功能。
